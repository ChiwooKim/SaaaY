## 프로젝트 소개

* 프로젝트명: 커뮤니티 음성대화 및 영상공유 서비스
* 서비스 특징: 간단하게 참여할 수 있는 주제별 대화 서비스
* 주요 기능
    - 회원 관리
    - 화상 및 음성 미팅룸
    - 그룹 채팅
    - 팔로우 / 차단
* 주요 기술
    - WebRTC
    - WebSocket
    - JWT Authentication
    - REST API
* 참조 리소스
    * Vuetify : 디자인 전반 적용
    * Vue Script, Vue.js Examples : 각종 컴포넌트 활용
    * Nicepage : View 디자인
* 배포 환경
    - URL: 마지막에
    - 테스트 계정: 구글로그인하세요
